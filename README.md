# Guess-The-Color
Guess the Color is a simple and fun web game built with HTML, CSS, and JavaScript. A random color appears, and you must choose the correct RGB value from three options. Get instant feedback and generate new colors to keep playing. Test your color-matching skills and improve your accuracy!
